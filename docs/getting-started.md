# Getting Started

## Installation

```bash
pip install peerport
```

Or with [uv](https://docs.astral.sh/uv/):

```bash
uv add peerport
```

## Basic Usage

```python
from peerport import add

result = add(1, 2)
print(result)  # 3
```
